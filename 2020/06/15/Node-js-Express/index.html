폴더 구조 (1Depth)

  • app.js
    핵심적인 서버 역할

  • bin/www
    서버를 실행하는 스크립트

  • public
    외부(브라우저 등의 클라이언트)에서 접근 가능한 파일들을 모아둔곳
    이미지, 자바스크립트, css 파일들이 있음

  • routes
    주소별 라우터들을 모아둔 곳

  • views
    템플릿 파일을 모아둔곳
