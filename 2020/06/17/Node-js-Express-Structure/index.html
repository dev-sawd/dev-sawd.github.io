[Node.js] 익스프레스 구조

6월 17, 2020
Node.js
  1. 1. bin/www
  2. 2. app.js

익스프레스의 각 폴더 구조에 대해서


bin/www

http 모듈에 express 모듈을 연결하고, 포트를 지정함


app, debug, http 모듈을 가져옴


app.set('port', port)로 서버가 실행될 포트 설정
precess.env객체에 PORT 속성이 있다면 그 값을 사용하고, 없으면 기본값으로 3000번 포트를 이용
app.set(키, 값)을 사용해서 데이터를 저장할 수 있고 app.get(키)로 가져올 수 있음


http.createServer에 불러온 app 모듈을 넣어줌
app 모듈이 createServer 메서드의 콜백 함수 역할을 함



app.js

express 패키지를 호출하여 app 변수 객체를 만듦
이 변수에 각종 기능을 연결


app.set메서드로 익스프레스 앱을 설정


app.use로 시작하는 코드는 미들웨어를 연결하는 부분


app 객체를 모듈로 만듦
bin/www에서 사용된 app이 이 객체 모듈
