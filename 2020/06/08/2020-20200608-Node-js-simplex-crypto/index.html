[Node.js] 단방향 암호화

6월 8, 2020
Node.js

비밀번호는 보통 단방향 암호화 알고리즘을 사용해서 암호화함
단방향 암호화 알고리즘은 주로 해시 기법을 사용
해시 기법이란 어떠한 문자열을 고정된 길이의 다른 문자열로 바꿔버리는 방식

createHash(알고리즘)

사용할 해시 알고리즘
md5, sha1, sha256, sha512등이 가능, 현재는 sha512정도로 충분

update(문자열)

변환할 문자열

digest(인코딩)

인코딩할 알고리즘
base64, hex, latin1이 주로 사용, 그중 base64 결과 문자열이 가장 짧아 애용
