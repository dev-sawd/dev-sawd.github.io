[CSS] Flexbox 정리

2월 21, 2022
CSS

프론트 개발자가 아니라 항상 처음부터 UI를 작성할때 어려움을 많이 느낌 복사 붙여넣기는 잘함

flexbox의 속성들을 간단히 정리

AXIS

main axis

사용자가 지정한 중심축

across axis

사용자가 지정한 중심축의 수직축

container, item

container

말그대로 다른 엘리먼트들을 담을 수 있는 컨테이너

container 속성

  • display
  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content(main axis)
  • align-items(across axis)
  • align-content(across axis)

item

컨테이너 안에 담긴 엘리먼트

item 속성

  • order
  • flex-grow
  • flex-shrink
  • flex
  • align-self

% vs vh

%는 부모 엘리먼트를 기준으로 x%를 채움
vh는 보이는 viewport height를 기준으로 채움
